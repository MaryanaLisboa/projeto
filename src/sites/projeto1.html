CONTEÚDOS

HTML5 é a versão mais recente da linguagem de marcação de hipertexto (HTML), usada para estruturar conteúdo web. Introduz novas tags, funcionalidades e APIs para páginas interativas.

CSS define como os elementos de uma página web são exibidos, incluindo cores, fontes, layout e outros aspectos visuais.

O que é a propriedade Display em CSS

A propriedade display em CSS define como um elemento HTML é exibido na página, determinando seu tipo de renderização e interação com outros elementos.

Display Características
Inline Exibe elementos em linha, lado a lado.
Inline-Block Exibe elementos em linha, mas permite definir largura e altura.
Block Exibe elementos como blocos, ocupando toda a largura disponível.
